#Project Name

Tic Tac Toe

## Description

Allows two players to play Tic Tac Toe on the command line. Can be Human vs Human, AI vs AI, or Human vs AI.

### Installation

Clone the repository onto desktop.

### Usage

Enter into the /lib folder and enter the following into a terminal:

```
ruby tic_tac_toe_game.rb
```
